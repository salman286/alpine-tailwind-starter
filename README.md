# Webpack, AlpineJS and TailwindCSS starter code

## Installing Dependencies.

Run following command

```sh
yarn
```

or using npm

```sh
npm install
```

## Start Development Server

Run following command

```sh
yarn dev
```

or using npm

```sh
npm run dev
```

## Make Build of Project

Run following command

```sh
yarn build
```

or using npm

```sh
npm run build
```
